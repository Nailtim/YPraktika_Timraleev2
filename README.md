# 📦 Система обслуживания вендинговых аппаратов (Vending Service System)

Десктопное приложение на языке **Java** для автоматизации процессов учета, технического обслуживания и диагностики парка вендинговых аппаратов. В качестве хранилища данных используется локальная встраиваемая база данных **SQLite**.

Интерфейс приложения выполнен в современном эргономичном **красно-черном стиле**, оптимизированном для долгой работы сервисного инженера.

---

## 🚀 Основные возможности (Функционал)

Программа разделена на три ключевых логических модуля:

### 1. Управление парком аппаратов
* **Добавление аппарата** — быстрая регистрация новых точек в системе (модель, адрес).
* **Просмотр списка** — удобная интерактивная таблица всех аппаратов.
* **Редактирование и удаление** — оперативное изменение данных или списание оборудования.
* **Обновление списка** — принудительная синхронизация данных с базой.

### 2. Техническое обслуживание (ТО)
* **Выбор аппарата для обслуживания** — быстрая фокусировка на конкретном объекте.
* **Добавление записи о ТО** — регистрация проведенных работ (инкассация, чистка, замена ингредиентов).
* **История обслуживания** — ведение и просмотр детального журнала работ по каждому аппарату с возможностью обновления данных.

### 3. Диагностика и Логирование
* **Ведение журналов операций** — автоматическая фоновая запись всех действий в файлы логов.
* **Просмотр журналов с инклудами** — встроенный просмотрщик логов, объединяющий связанные файлы.
* **Открытие папки с логами** — быстрый переход к директории с файлами логов в проводнике ОС.
* **Консоль отладки** — вывод технических сообщений системы в реальном времени.
* **Монитор производительности** — графическое отображение нагрузки на ЦП (CPU) и ОЗУ (RAM) текущего ПК.

---

## 🛠 Технологический стек

* **Язык программирования:** Java (JDK 11+)
* **Интерфейс пользователя (UI):** JavaFX / Java Swing (в красно-черной цветовой гамме)
* **База данных:** SQLite (встраиваемая, файл `vending.db`)
* **Подключение к БД:** JDBC Driver (SQLite-JDBC)

---

## 📂 Структура базы данных (`vending.db`)

Локальная база данных состоит из двух основных таблиц:
1. `vending_machines` — информация об аппаратах (ID, модель, адрес, статус).
2. `service_history` — история обслуживания (ID записи, ID аппарата, дата, описание работ).
